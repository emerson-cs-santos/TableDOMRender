Gerador de tabela em HTML com base em um JSON

Gerar Tabela

Insira um JSON válido abaixo para gerar uma tabela responsiva em HTML utilizando bootstrap. É aceito apenas JSON simples (Ou seja, JSON dentro de JSON será ignorado), clique no botão de exemplo para ver um.

Os nomes das colunas serão os nomes dos campos que estirem no JSON. Os elementos serão adicionados na arvore DOM.
